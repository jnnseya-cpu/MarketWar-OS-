## Extracted findings (structured — from the summary above)

**The mirror-image gap:**
- **BLG** — creates optimised content · builds backlinks via a **4,000+ site network** · tracks AI visibility on autopilot · **but never listens**.
- **Brandwatch** — **1.4 trillion conversations, 500M+ added daily** · **but never executes**.
- **Convergence signal:** Brandwatch's 2026 roadmap adds **GEO measurement** (how brands get recommended in ChatGPT / Perplexity) → the categories are merging. MarketWar OS plants its flag at the intersection: **listen + execute + get recommended by AI engines**, in one OS.

**Two asymmetric advantages neither incumbent can copy:**
1. **Portfolio-blog cold-start solve** — the owner's existing portfolio blogs give the backlink/authority network a day-one seed, defeating the cold-start problem BLG's 4,000-site network needed years to build.
2. **Africa Theatre Pack** — French / Lingala / Swahili listening + **DRC publisher inventory** + **BitriPay attribution data**. Both incumbents are absent: BLG lists 50+ languages, **zero African**; Brandwatch's DRC-relevant channel coverage is **effectively nil**. (Ties directly to the v3 spec's Phase-4 African-diaspora corridor, doc 08 §C.3, and BitriPay, doc 05.)

**The wedge against their weakness:**
- Brandwatch's Iris assumes **analysts who speak Boolean**; without them it feels like an expensive data warehouse.
- **MW-07 Query Compiler** eliminates that: natural-language → Boolean listening query, so non-analysts get analyst-grade listening. (Natural sibling to the shipped SiteRaid AI approach — plain input, expert output.)
